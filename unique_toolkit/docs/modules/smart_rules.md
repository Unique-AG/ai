# Smart Rules Module

The Smart Rules module provides functionality for compiling and working with smart rules (UniqueQL queries).

## Overview

The `unique_toolkit.smart_rules` module is deprecated. Please use `unique_toolkit.content.smart_rules` instead.

Smart rules allow you to create complex queries using UniqueQL to filter and search content in the knowledge base.

## Deprecation Notice

This module is deprecated. Use `unique_toolkit.content.smart_rules` for all smart rules functionality.
